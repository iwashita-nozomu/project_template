# From Another Agent

1. Read `notes/themes/from_another_agent.md` before planning.
1. Treat `## やるべきこと` as carry-over backlog unless the user overrides it.
1. Use `## お勧め機能` only when the adjacent work is still within the current task scope.
1. When an item is resolved, superseded, or promoted to canon, update the note in the same change.
1. Keep durable rules in `agents/` or `documents/`; keep the note short and task-oriented.
